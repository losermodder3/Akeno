## fazer a rias funfar use os comando:

```bash
> termux-setup-storage
> pkg install git
> pkg install ffmpeg
> pkg install wget
> pkg install nodejs

> cd Rias
> bash install.sh
```

```bash
> npm start
```

```bash
> depois = pm2 monit
```
